Comment créer un compte à rebours en utilisant JavaScript

Laisser un commentaire / GBlog, Technical Scripter / Par Acervo Lima

Un compte à rebours est une minuterie précise qui peut être utilisée pour un site Web ou un blog pour afficher le compte à rebours jusqu’à tout événement spécial, tel qu’un anniversaire ou un anniversaire.
Les bases d’un compte à rebours sont :

  1. Définissez une date de fin valide.
  2. Calculez le temps restant.
  3. Convertissez l’heure dans un format utilisable.
  4. Sortez les données d’horloge en tant qu’objet réutilisable.
  5. Affichez l’horloge sur la page et arrêtez l’horloge lorsqu’elle atteint zéro.

Étape 1 : Définir une date de
fin valide La date et l’heure de fin valide doivent être une string dans l’un des formats compris par la méthode Date.parse() de JavaScript.

Étape 2 : Calculer
le temps restant Nous calculons d’abord le temps restant en soustrayant la date limite par la date et l’heure actuelles puis nous calculons le nombre de jours, heures, minutes et secondes. La fonction Math.floor() est utilisée pour renvoyer le plus grand entier moins supérieur ou égal à un nombre donné.

Étape 3 : Sortir le résultat
Dans le code ci-dessous le résultat est donné en sortie par id= »demo »

Étape 4 : Écrivez du texte si le compte à rebours est terminé
Si le compte à rebours est terminé, alors « expiré » sera affiché à l’écran.

SAISIR :

<!DOCTYPE HTML>
<html>
<head>
<style>
p {
  text-align: center;
  font-size: 60px;
}
</style>
</head>
<body>
<p id="demo"></p>
<script>
var deadline = new Date("Jan 5, 2018 15:37:25").getTime();
var x = setInterval(function() {
var now = new Date().getTime();
var t = deadline - now;
var days = Math.floor(t / (1000 * 60 * 60 * 24));
var hours = Math.floor((t%(1000 * 60 * 60 * 24))/(1000 * 60 * 60));
var minutes = Math.floor((t % (1000 * 60 * 60)) / (1000 * 60));
var seconds = Math.floor((t % (1000 * 60)) / 1000);
document.getElementById("demo").innerHTML = days + "d " 
+ hours + "h " + minutes + "m " + seconds + "s ";
    if (t < 0) {
        clearInterval(x);
        document.getElementById("demo").innerHTML = "EXPIRED";
    }
}, 1000);
</script>
  
</body>
</html>

Applications du compte à rebours


    

  • Utilisé pendant les événements pour afficher le temps restant pour son début.
    • 

  • Utilisé par les sites de commerce en ligne pour afficher le temps restant pour une vente en cours.
    • 

  • Utilisé par les sites Web lors des promotions
    • 

  • Utilisé dans les jeux de course automobile, les jeux de football, etc.
    • 

  • Utilisé dans les sites Web d’enchères pour afficher la gauche pour placer des offres.
    • 



Avantages de faire un compte à rebours en JavaScript que d’utiliser des plugins


    

  • Le code sera léger car il n’aura aucune dépendance.
    • 

  • Le site Web fonctionnera mieux car il n’y aura pas besoin de charger des scripts externes et des feuilles de style.
    • 

  • L’utilisateur obtient plus de contrôle car il a construit l’horloge pour qu’elle se comporte exactement comme il le souhaite plutôt que d’essayer de plier un plugin selon sa volonté.
